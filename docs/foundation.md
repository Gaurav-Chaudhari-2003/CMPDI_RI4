# Application Foundation & Governance

This document outlines the core architectural principles, governance model, and development workflow for the CMPDI RI-4 application. All contributors are expected to adhere to these guidelines to ensure the project remains scalable, maintainable, and secure.

## Foundation Document

The foundational principles and high-level system architecture for this project are detailed in the official CMPDI RI-4 System Architecture document. This document serves as the single source of truth for the project's technical and strategic direction.

- **[View the Foundation Document](/docs/CMPDI-RI-4-System-Architecture.pdf)**

Before contributing, please familiarize yourself with the concepts outlined in this document, particularly concerning:

-   Domain-Driven Design (DDD) structure
-   Role-Based Access Control (RBAC) implementation
-   API boundary and communication guidelines
-   Coding standards and best practices

Adherence to this document is mandatory for all development work.
